# Концепт к Уроку 1 Django "Пишем Яндекс.Афишу"

```yaml
Активный отдых: !concept
  определение: Активный отдых - минипутешествия, посещение достопримечательностей и необычных мест.
  Японский сад: !extension |
    Ландшафтная экспозиция «Японский сад» Главного ботанического сада, открыта для посещений с конца апреля до середины октября.
  Павильон "Космос" на ВДНХ: !extension |
    Музейный комплекс, посвященный отечественной авиации и истории освоения космоса.
  Останкинская башня: !extension |
    Телевизионная и радиовещательная башня, расположенная в Останкинском районе Москвы.

Контент сайта: !concept
  определение: Информационное содержимое интернет-ресерса.
    Географические координаты: !extension |
      Координаты объекта на Яндекс картах для определения его местоположения.
    Текстовое описание: !extension |
      Краткое и подробное описания особенностей объекта.
    Фотографии: !extension |
      Фотографии, дающие визуальное представление об объекте.

Разработчики сайта: !concept
  определение: Программисты, разрабатывающие сайт и администраторы, отвечающие за его заполнение.
    Фронтендер:  !extension |
      Специалист, разрабатывающий веб интерфейс и отвечающий за визуальную составляющую сайта.
    Бэкендер:  !extension |
      Специалист, отвечающий за серверную часть кода, за то, как сайт будет реагировать на действия пользователя.
    Администратор: !extension |
      Специалист, отвечающий за заполнение сайта, его функционирование.
```

# Роли

```yaml
Администратор сайта: !role
  Мотивация:
    - Наполнять сайт актуальной и интересной информацией.
  Старт: Админка сайта Яндекс.Афиша "Интересные места Москвы"
  
Бэкендер: !role
  Мотивация:
    - Обеспечить функциональность и безошибочную работу сайта.
  Старт: Гитхаб проекта Яндекс.Афиша "Интересные места Москвы"
  
Фронтендер: !role
  Мотивация:
    - Разработать красивый и информативный интерфейс.
  Старт: Гитхаб проекта Яндекс.Афиша "Интересные места Москвы"
```

# Истории

```yaml
Пользователь сайта - найти новое интересное место: !func
  Найти ближайшее новое интересное место: !story
    ситуация:
      - уже посещал
      - не посещал
    старт: Клик по ближайшей к интересующему адресу красной точке, находящейся на карте, размещенной на сайте проекта
    успех:
      - Справа от карты появилось описание и фотографии объекта.
      - Объект ранее не посещался

Бэкенд-программист - подключиться к виртуальному окружению для работы над проектом: !func
  Использовать систему управления пакетами и средами conda для входа: !story
    ситуация:
      - conda не установлена на компьютер программиста
      - conda установлена виртуальное окружение создано
      - conda установлена, виртуальное окружение не создано
    старт: Запуск команды активации виртуального окружения
    успех:
      - в терминале перед приглашением в скобочках есть название виртуального окружения
    что понадобится:
      - Бэкенд-программист - установить conda
      - Бэкенд-программист - создать виртуальное окружение

Бэкенд-программист - создать виртуальное окружение: !func
  Использовать систему управления пакетами и средами conda для создания окружения: !story
    ситуация:
      - conda не установлена на компьютер программиста
      - conda установлена
    старт: Запуск команды создания виртуального окружения
    успех:
      - появилось сообщение о создании виртуального окружения
      - можно активировать созданное виртаульное окружение
    что понадобится:
      - Бэкенд-программист - установить conda

Бэкенд-программист - установить conda: !func
  Установить скачанный пакет: !story
    старт: Запуск команды установки
    успех:
      - появилось сообщение об успешной установке

Системный администратор - загрузить в базу контент: !func
  Используя команду загрузки заполнить базу: !story
    ситуация:
      - известена веб-ссылка, где расположена информация для загрузки
      - имеется в наличии подготовленный файл для загрузки
    старт: Запуск команды в терминале на сервере с проектом и подключенным виртуальным окружением
    успех:
      - в терминале отобразится результат загрузки
      - на сайте появятся объекты посещения и информация о них
    отказ:
      - у случае ошибки в терминале появится сообщение об этом
  

```
